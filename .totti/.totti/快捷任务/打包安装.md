# 任务：PhonePad 打包安装

## 项目信息

| 项 | 值 |
|---|---|
| 项目名称 | PhonePad |
| 本地路径 | `E:\totti\phonepad\phonepad` |
| 当前版本 | 见 `package.json`（如 `0.2.0`） |
| 桌面端安装目录 | `%LOCALAPPDATA%\PhonePad Receiver\` |
| Android 包名（Debug） | `cn.phonepad.debug` |
| Android 包名（Release） | `cn.phonepad` |

---

请为本机构建 PhonePad 桌面端与 Android 端最新版本，并安装到当前电脑与 adb 已连接的手机。

## 必须严格执行以下全部流程

### 1. 进入项目目录

```powershell
cd E:\totti\phonepad\phonepad
```

> Git 仓库根目录是 `E:\totti\phonepad\phonepad`，不是上级 `E:\totti\phonepad`。

### 2. 构建桌面端（Release）

```powershell
cd E:\totti\phonepad\phonepad
npm run lint
npm run desktop:build
```

构建产物：

| 类型 | 路径 |
|------|------|
| 可执行文件 | `target\release\phonepad-desktop.exe` |
| NSIS 安装包 | `target\release\bundle\nsis\PhonePad Receiver_<版本>_x64-setup.exe` |

### 3. 构建 Android APK（Debug）

```powershell
cd E:\totti\phonepad\phonepad
npm run android:assemble
```

构建产物：

| 类型 | 路径 |
|------|------|
| Debug APK | `apps\android\app\build\outputs\apk\debug\app-debug.apk` |

Release 包（可选）：

```powershell
npm run android:assemble:release
```

产物路径：`apps\android\app\build\outputs\apk\release\app-release-unsigned.apk`

### 4. 安装桌面端到本机

安装前先结束旧进程，再静默安装 NSIS 包：

```powershell
Get-Process *phonepad*,*totti* -ErrorAction SilentlyContinue | Stop-Process -Force -ErrorAction SilentlyContinue
Start-Sleep -Seconds 1

$setup = Get-ChildItem "E:\totti\phonepad\phonepad\target\release\bundle\nsis\PhonePad Receiver_*_x64-setup.exe" | Select-Object -First 1
Start-Process -FilePath $setup.FullName -ArgumentList "/S" -Wait
```

验证安装：

```powershell
Get-Item "$env:LOCALAPPDATA\PhonePad Receiver\phonepad-desktop.exe" | Format-List FullName, LastWriteTime, Length
tasklist /FI "IMAGENAME eq phonepad-desktop.exe"
```

可选：启动桌面端

```powershell
Start-Process "$env:LOCALAPPDATA\PhonePad Receiver\phonepad-desktop.exe"
```

### 5. 安装 Android APK 到 adb 设备

检查设备：

```powershell
adb devices
```

安装 Debug APK（覆盖安装）：

```powershell
adb install -r "E:\totti\phonepad\phonepad\apps\android\app\build\outputs\apk\debug\app-debug.apk"
```

多设备时指定序列号：

```powershell
adb -s <设备序列号> install -r "E:\totti\phonepad\phonepad\apps\android\app\build\outputs\apk\debug\app-debug.apk"
```

### 6. 安装完成后展示结果

必须包含：

- 桌面端：安装路径、文件时间、进程是否运行
- Android：adb 设备列表、安装命令 exit code
- 本次构建版本号（来自 `package.json`）
- 若构建或安装失败，说明原因与下一步
